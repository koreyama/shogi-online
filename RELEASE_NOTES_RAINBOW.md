# Rainbow Game Release Notes

## 🌈 新ゲーム「Rainbow」正式リリース

UNOスタイルのカードゲーム「Rainbow」が実装されました。
色と数字を合わせて手札を減らし、勝利を目指しましょう！

### ✨ 主な機能

*   **クロス・プレイヤー・レイアウト**:
    *   プレイヤーを十字型（3x3グリッド）に配置し、時計回り・反時計回りの順番が直感的に分かるようになりました。
    *   中央の回転インジケーターで手番の進行方向を可視化。

*   **快適な手札操作**:
    *   **ドラッグスクロール**: 手札が増えても、マウスやタッチ操作で直感的にスクロール可能です。
    *   **スマートな表示**: 自分の手番では手札がハイライトされ、操作しやすくなります。
    *   **ソート機能**: 「🎨 色順」「123 数字順」ボタンで、サーバーサイドで確実に手札を整頓できます。

*   **戦略的なカードプレイ**:
    *   **複数枚出し**: 同じ数字のカードを一度に複数枚出すことができます（出す順番も指定可能）。
    *   **カウンター**: ドロー2やワイルド4に対して、さらにドローカードを重ねて回避・攻撃するカウンタープレイが可能です。

*   **モバイル最適化**:
    *   スマートフォンでも快適に遊べるよう、レイアウトと操作エリアを調整しました。
    *   誤操作防止のための安全なパディング確保。

### 🛠️ 技術的な改善点

*   **サーバーサイド・ソート**: 手札の並び替えをサーバーで処理することで、クライアント間の同期ズレを防止。
*   **堅牢なスクロール実装**: CSS FlexboxとJavaScriptによるカスタムドラッグロジックで、あらゆるデバイスでの操作性を保証。
*   **Colyseus & Firebase連携**: リアルタイム通信とロビー機能をシームレスに統合。
